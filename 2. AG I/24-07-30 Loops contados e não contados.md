## `while()`
### Contado

> [!NOTE] O que precisa
> 1. Variável de controle
> 2. Teste para o fim
> 3. Incremento / decremento
```python
cont = 0           # Variável de controle
while(cont < 10):  # Teste
	print(cont)
	cont += 1      # Incremento
```
### Não Contado

> [!NOTE] O que precisa
> 1. Variável de controle
> 2. Teste para o fim
```python
cont = 0           # Variável de controle
while(cont < 10):  # Teste
	print(cont)
	cont += 1      # Incremento
```
